# Scripts Directory

Utility scripts for SouCul project setup and maintenance.

## Database Testing

### test-db-connection.js
Tests database connection using credentials from `.env` file.

**Usage:** `node scripts/test-db-connection.js`

### diagnose-db.js
Diagnostic tool for database setup issues.

**Usage:** `node scripts/diagnose-db.js`

### run-db-test.bat
Quick Windows shortcut for running the connection test.

**Usage:** `scripts\run-db-test.bat`
